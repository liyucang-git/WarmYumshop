# 暖圆小铺

暖圆小铺是一款聚焦家庭场景的轻量工具小程序，帮助家庭记录和分享私房菜谱，留住美食时光。

## 功能特性

- **用户认证**：使用微信登录，快速便捷
- **家庭管理**：创建家庭、加入家庭、查看家庭信息
- **菜品管理**：添加、编辑、删除菜品
- **图片上传**：支持菜品图片上传和预览
- **分类管理**：多种菜品分类，方便管理
- **响应式设计**：适配不同屏幕尺寸

## 技术栈

- **前端**：微信小程序原生开发
- **后端**：微信云开发（云函数、云数据库、云存储）
- **UI组件**：微信原生组件
- **开发工具**：微信开发者工具

## 安装和运行

### 前置条件

- 微信开发者工具
- 微信小程序账号
- 云开发环境

### 步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   ```

2. **打开项目**
   - 打开微信开发者工具
   - 选择「导入项目」
   - 选择项目目录 `WarmYumshop`
   - 填写小程序 AppID

3. **配置云开发环境**
   - 在微信开发者工具中，点击「云开发」按钮
   - 按照提示创建云开发环境
   - 在 `app.js` 文件中，确保云开发初始化配置正确

4. **部署云函数**
   - 在微信开发者工具中，右键点击 `cloudfunctions` 目录
   - 选择「上传并部署所有云函数」

5. **运行项目**
   - 点击微信开发者工具中的「编译」按钮
   - 在模拟器中查看效果
   - 也可以通过「预览」功能在真机上查看

## 核心功能说明

### 1. 用户认证
- 使用微信登录，获取用户信息
- 登录状态持久化存储

### 2. 家庭管理
- 创建家庭，生成邀请码
- 通过邀请码加入家庭
- 查看家庭信息和成员

### 3. 菜品管理
- 添加新菜品，包括名称、分类、描述和图片
- 编辑现有菜品信息
- 删除不需要的菜品
- 查看菜品详情

### 4. 图片上传
- 支持从相册选择或拍照
- 图片上传到云存储
- 图片预览功能

## 注意事项

1. **云开发环境**：确保已创建并配置好云开发环境
2. **云函数部署**：首次运行前需要部署所有云函数
3. **权限设置**：确保云存储权限设置正确，允许上传和访问图片
4. **用户登录**：使用前需要先登录微信
5. **家庭加入**：添加菜品前需要先加入或创建家庭
